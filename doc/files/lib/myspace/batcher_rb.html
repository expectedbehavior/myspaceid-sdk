Last Update: Wed Apr 15 13:54:11 -0700 2009

class for batching up changes and submitting them at preset intervals

size indicates how many to accumulate init initializes whatever data structure you‘re using to store the changes flusher is a block that gets every ‘size’ times ‘push’ gets called

call push to add something to the batch and call flush when you‘re done adding stuff
